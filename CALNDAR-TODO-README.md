# Calendar + To Do Page

## Overview
This page displays your weekly calendar events and todo list in a split view:
- **Left (2/3)**: Apple Calendar events for the next three days (auto-generated by `scripts/auto-update-data.js`)
- **Right (1/3)**: Todo items managed by the Chat agent

## Files
- `app/calendar-todo/page.tsx` - The main page component
- `app/api/calendar-todo/route.ts` - API endpoint that reads the generated JSON files
- `data/calendar.json` - Cached calendar events for the template
- `data/todos.json` - Todo items captured from chat conversations
- `scripts/auto-update-data.js` - Auto-update script (AppleScript + ICS fallback + blob upload)
- Blob backup: `calendar/events.json` (kept in Vercel Blob for downstream use)

## Auto-Update Schedule
The data is refreshed four times daily (America/New_York):
- **9:00 AM** — morning batch
- **12:00 PM** — lunchtime snapshot
- **6:00 PM** — evening checkpoint
- **12:00 AM** — midnight recap

The cron jobs run the same `scripts/auto-update-data.js` command and announce summaries into the Discord #chat/#claw channels.

## How the script works
1. Uses `osascript` (JavaScript for Automation) to pull the next 3 days of events from Apple Calendar.
2. Falls back to a local `calendar-export.ics` if Automation is blocked.
3. Extracts todo bullets from `memory/YYYY-MM-DD.md` and `CHAT-TODO.md`.
4. Writes `data/calendar.json` and `data/todos.json`.
5. Uploads the calendar slice to Vercel Blob at `calendar/events.json` (requires `BLOB_READ_WRITE_TOKEN`).

Make sure the machine running the script has Automation access to Calendar (approve the prompt for `node`/`osascript`).
