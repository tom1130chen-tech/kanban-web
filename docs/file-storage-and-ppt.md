# 📁 文件存储与 PPT 生成指南

## 1. 文件存储方案

### ⚠️ Vercel Blob 用途限制

**仅用于**：
- 网站内容（newsletter、board-state 等）
- 需要公开访问的数据

**不用于**：
- 个人文件存储
- 大文件传输
- 临时文件交换

---

### ✅ 推荐方案：GitHub Releases

**优势**：
- ✅ 免费，每个文件最大 2GB
- ✅ 已集成，无需额外配置
- ✅ 支持私有仓库
- ✅ 版本管理
- ✅ 永久存储

**使用流程**：

```
你上传文件给我
    ↓
我处理/分析内容
    ↓
保存到 workspace
    ↓
Commit 到 GitHub
    ↓
创建 Release 提供下载
    ↓
返回下载链接给你
```

**示例命令**：
```bash
# 创建 Release（需要 GitHub CLI）
gh release create v1.0.0 file.pptx --title "PPT Download" --notes "Generated by Chat"
```

---

### 备选方案：OneDrive

**优势**：
- 5GB 免费空间
- 支持大文件
- 熟悉的界面

**劣势**：
- 需要 OAuth 认证配置
- API 较复杂
- 需要定期刷新 token

**配置步骤**（如果需要）：
1. 注册 OneDrive API
2. 获取 Client ID 和 Secret
3. 配置 OAuth 流程
4. 存储 refresh token

---

## 2. PPT 生成

### ✅ 已安装库

- `python-pptx` v1.1.2
- 位置：系统 Python 环境

### 📝 使用方法

#### 方法 A：提供内容，我生成 PPT

```
你：给我一个主题和要求
我：生成内容大纲 → 创建 PPT → 上传到 GitHub Releases → 给你链接
```

#### 方法 B：使用 JSON 模板

1. **创建 JSON 文件**：

```json
{
  "title": "你的演示标题",
  "slides": [
    {
      "title": "第一页标题",
      "content": [
        "要点 1",
        "要点 2",
        "要点 3"
      ],
      "notes": "演讲者备注（可选）"
    }
  ]
}
```

2. **运行脚本**：

```bash
cd /Users/tomchen/.openclaw/workspace-chat/kanban-board
python3 scripts/create-ppt.py
```

## 4. 文件传输流程

### 你发送文件给我

**支持格式**：
- ✅ PDF
- ✅ Word (.docx)
- ✅ PowerPoint (.pptx)
- ✅ Excel (.xlsx)
- ✅ 图片 (JPG, PNG)
- ✅ 文本文件

**限制**：
- Discord 渠道：最大 10MB
- 更大文件：分卷或压缩

### 我返回文件给你

**方式 1：GitHub Releases**（推荐）
```
1. 我创建 Release
2. 上传文件
3. 返回链接：https://github.com/tom1130chen-tech/kanban-web/releases/tag/v1.0.0
4. 你下载
```

**方式 2：Blob 临时存储**（仅限小文件）
```
1. 上传到 Blob
2. 返回公开链接
3. 24 小时后自动清理
```
